Linux mkisofs命令：制作ISO映像文件

mkisofs 是一个多功能命令行实用程序，用于从目录创建 ISO 映像文件。ISO文件是光盘映像文件，可用于分发大量文件、进行备份或创建可引导媒体。该命令也称为 genisoimage 。

创建ISO映像文件

mkisofs -o filename.iso path/to/source_directory

选项说明

  • -o filename.iso：该选项指定输出文件名。在此示例中，最终输出的ISO文件将命名为：filename.iso
  • path/to/source​_directory：这表示其内容将包含在 ISO 映像中的目录。该路径应替换为您要存档的目录的实际路径。

创建ISO映像文件时指定名称

mkisofs -o filename.iso -V "labelname" path/to/source_directory

选项说明

  • -o filename.iso：该选项指定输出文件名。在此示例中，最终输出的ISO文件将命名为：filename.iso
  • -V "labelname"：该选项设置 ISO 的卷名称或光盘标签，为映像提供可识别的名称。
  • path/to/source​_directory：这表示其内容将包含在 ISO 映像中的目录。该路径应替换为您要存档的目录的实际路径。
