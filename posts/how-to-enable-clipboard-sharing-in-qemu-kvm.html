如何在QEMU/KVM虚拟机上共享剪切板/支持主机和虚拟机间的复制粘贴

Table of Contents

  • 1. 简介
  • 2. Windows虚拟机
  • 3. Ubuntu虚拟机
  • 4. CentOS虚拟机

1. 简介

使用QEMU虚拟机时，在主机和虚拟机之间共享剪切板是一项非常方便的特性。

本文将介绍如何实现这一功能。

2. Windows虚拟机

Windows虚拟机可参考以下方法：

  1. 在虚拟机里访问此链接 SPICE。
  2. 在打开的网页中，向下滚动，找到 Guest 章节，在 Windows Binaries 中下载 Windows SPICE Guest Tools。
  3. 在虚拟机中安装 Windows SPICE Guest Tools 。

3. Ubuntu虚拟机

Ubuntu系列的虚拟机使用以下命令： 

sudo apt install -y qemu-guest-agent
sudo apt install -y spice-vdagent

4. CentOS虚拟机

CentOS系列的虚拟机使用以下命令： 

sudo yum install -y qemu-guest-agent
sudo yum install -y spice-vdagent
