Linux系统中查找域名IP地址的方法

简介

在Linux系统中，可以使用以下 5 个命令查找域名IP地址：

  • dig 命令：它是一个用于查询 DNS 名称服务器的灵活命令行工具。
  • host 命令：它是用于执行 DNS 查询的简单程序。
  • nslookup 命令：它用于查询互联网域名服务器。
  • ping 命令：它用于向网络主机发送 ICMP ECHOREQUEST 数据包。
  • wget命令：它是一个从网络上下载文件的工具。

方法一：dig

$ dig bing.com

方法二：host

$ host bing.com
bing.com has address 13.107.21.200
bing.com has address 204.79.197.200
bing.com has IPv6 address 2620:1ec:c11::200
bing.com mail is handled by 10 bing-com.mail.protection.outlook.com.

方法三：nslookup

~$ nslookup bing.com
